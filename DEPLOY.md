# Deployment Guide

Orbit Admin can be deployed as a static documentation site using GitHub Pages.

## Steps
1. Go to **Settings → Pages** in this repository.
2. Under **Source**, select `main` branch and `/docs` folder.
3. Save.  
4. Your site will be live at: `https://Gyanankur23.github.io/orbit-admin`

## Optional
- Add a custom domain under Pages settings.
- Use Netlify or Vercel for advanced deployment.
