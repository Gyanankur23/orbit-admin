## Overview
Orbit Admin is a **documentation-first admin framework** built entirely in Markdown.  
It provides a structured way to manage admin guides, changelogs, and contribution rules without requiring code.

## Business Use Case
- 📑 **Onboarding**: Clear admin docs for new hires
- 🔒 **Compliance**: Transparent changelogs for audits
- 🤝 **Collaboration**: Contributor guidelines for teams

## Deployment
Orbit Admin can be deployed as a static documentation site:
1. Fork or clone the repo
2. Enable **GitHub Pages** in repo settings
3. Your documentation will be live at: `https://<username>.github.io/orbit-admin`
